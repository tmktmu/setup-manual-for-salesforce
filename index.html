1. Visual Studio Code

Salesforceで、ソース駆動開発を行う際に利用するエディタ
拡張機能を追加することにより、複雑なノーコード設定からシンプルなプログラム開発を行うことができるように！

  • Visual Studio Codeをインストール
    ご自身のOSに合わせてダウンロード

    DownLoadはこちらをクリック

  • ダウンロード後実行ファイルをダブルクリック

  • 画面の指示に従って「次へ」を押下 最後の画面で「インストール」実行
    毎回すぐに開けるように以下の設定にチェックすること推奨

2. Salesforce CLI

ソース駆動開発やメタデータ管理を行うために必要
Visual Studioコードと併せて使用

  • Salesforce CLIをインストール
    ご自身のOSに合わせてダウンロード

    DownLoadはこちらをクリック

  • ダウンロード後実行ファイルをダブルクリック

  • 画面の指示に従って「次へ」を押下 最後の画面で「インストール」実行
    警告が表示されたら「はい」をクリック

3. Salesforce Extension Pack

Visual Studioコードにインストールする拡張機能

  • Salesforce Extension Pack
    Visual Studioコードを開いた状態で赤枠のアイコンを押下
    「Salesforce Extension Pack」

  • 「インストール」実行
